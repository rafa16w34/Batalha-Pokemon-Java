Mecânicas disponíveis:
- Usar ataques dos pokémons;
- Usar poções para curar um pokémon;
- O pokémon pode usar evasiva e desviar de um ataque, caso a velocidade dele for maior que do adversário ele sempre irá desviar, se não ele tem 50% de chance de desviar;
- Pikachu e Geodude só podem evoluir se tiverem seus itens de evolução;
- Mega evoluções para Raichu e Gyarados se tiverem conseguido as mega stones;
- Arceus pode mudar seu tipo ao usar o ataque Judgment;
